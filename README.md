# Customs Portal – Releases

This repository contains the official releases for **Customs Portal**, a desktop application for automating Romanian customs declaration workflows.

## What is Customs Portal?

Customs Portal is a Windows desktop application that automates the downloading of import (DVI) and export (DVE) customs declarations from the Romanian customs system. It eliminates manual, repetitive work by running browser automation in the background and streaming live progress directly in the interface.

## Key Features

- **Automated DVI & DVE downloads** – run declaration downloads with a single click
- **Live log streaming** – monitor progress in real time as the automation runs
- **Resume from last position** – if a run is interrupted, it resumes exactly from the last saved page and row
- **Auto-updates** – the application checks for new versions on startup and updates automatically

## Installation

1. Download the latest installer from the [Releases](../../releases/latest) page
2. Run `Customs Portal Setup x.x.x.exe`
3. Follow the on-screen instructions
4. Launch **Customs Portal** from the desktop shortcut

## Updates

Updates are delivered automatically. When a new version is available, the application will notify you and install it on the next restart.

## Requirements

- Windows 10 or later
- Active Customs Portal subscription
